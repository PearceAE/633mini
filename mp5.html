For Mini Project 5 is used Dreamweaver to polish my Mini Project pages. I started by creating my navigation bar in Mini Project 1, using the same color scheme that I have previously used. Once I had my table created, I then went to the page properties and set my link colors and underlining preferences. Once I had the navigation table set how I wanted it, I then deleted the previously created "Assignment List" as it would have been redunant with the navigation bar put in place. I then also moved my contact information to the navigation bar and decided to use a evenvelope image to add an additional visual element. I retreived the evenlope image from Google and then filtered my search for images that were available for reuse. Once I completed that, I then went through and added my anchor. I copied the navigation and anchor to each of my other Mini Project pages and then changed the links were appropriate (i.e. back to top anchor). Once each page looked correct visually, I then put each page through the HTML validator and corrected any errors that were displayed.
